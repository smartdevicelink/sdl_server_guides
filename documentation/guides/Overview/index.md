# Overview
This document provides the information for creating and integrating the SmartDeviceLink (SDL) server component with the mobile libraries and vehicle's Head Unit (HU).

The Policy Server's main purpose is to curate [policy tables](/docs/sdl-server/master/policy-table/overview) composed of rules and permissions defined by a vehicleâ€™s OEM. Each vehicle will download its Policy Table and use it to govern SDL behaviors.
